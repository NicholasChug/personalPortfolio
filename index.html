Project Showcase

Three-Page PSD Conversion

A three-page PSD conversion project. Built using HTML5 and SASS, this project was provided by a client to be built with all assets included. This project was styled specifically with flexbox, allowing for full responsiveness from webpage down to mobile devices.

CNTDWN

A simple countdown timer built using vanilla JavaScript with jQuery. A user is sugested to input a future date, and a timer will be revealed counting down to the specific date entered. Upon completion, alerts are utilized to notify the user. This web application was built to be fully responsive and accessible.

Game Night!

Game Night is an interactive and randomly generated quiz game. Utilizing JavaScript. jQuery and the use of the OpenTrivia API, this project was built alongside two collegues Ana Morales and Mohamed Omar. With the use of a selector, different categories of questions will be generated for the user.

Binge-Fest

Binge Night is a React web application built alongside three other collegues, Dani Shen, Natalia Andreola and Auschvin Vasanth. Using React.js along with the Zomato API and TVMaze Api, the app allows users to search either local restaurants with a keyword, or available tv shows. Once searched, the user can save these results and eventually ask the program to randomlly chose from the saved lists, allowing the user to have their evenings activities chosen for them.

today.

A to-do style web application built using React.js and firebase. The user can add and remove tasks from a dynamically created list. The data is stored using a Firebase database so the saved items persists upon page reloading.
